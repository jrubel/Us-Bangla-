# US Bangla Morning Load Project

A specialized flight load management system for US Bangla Airlines.

## Access
- **Default Password**: `admin123`
- If a custom password is required, set the `VITE_APP_PASSWORD` environment variable.

## Key Features
- **Dynamic Route Management**: Real-time ETA and duration calculation.
- **Multi-Sector Leg Creation**: Support for connecting flights via CGP for Middle East routes (DXB, AUH, MCT, DOH).
- **Paired Flight View**: Automated return leg creation with flight number sequencing logic.
- **Telemetry Charts**: Visual analysis of passenger loads across sectors.
- **Cyberpunk UI**: High-contrast, tech-inspired interface for mission-critical operations.
